# Welcome to VoidRun

VoidRun is a modern sandbox environment that provisions highly secure, isolated MicroVMs for AI agents and developers. Execute code safely with enterprise-grade security and lightning-fast performance.

## What is VoidRun?

VoidRun provides isolated MicroVM environments designed for AI workloads and developer tooling. Whether you're building AI agents, code interpreters, or CI/CD pipelines, VoidRun offers the security and flexibility you need.

### Key Features

<CardGroup cols={3}>
  <Card title="⚡ Performance" icon="bolt">
    Infinite running sandboxes with auto-sleep after 5 minutes of idle time. Resume in under 200ms.
  </Card>
  <Card title="🐳 Flexible Environments" icon="container-storage">
    Custom Docker images supported. Run Docker and K3s natively inside sandboxes.
  </Card>
  <Card title="🔒 Security" icon="shield-lock">
    Granular internet access control with allowlists and blocklists. Native Tailscale VPN connectivity.
  </Card>
  <Card title="💻 Code Execution" icon="terminal">
    Full code interpreter capabilities. Install Ubuntu, Python, and TypeScript packages dynamically.
  </Card>
  <Card title="🌐 Direct Access" icon="arrow-right-arrow-left">
    Interact with sandboxes via VNC and SSH for debugging and development.
  </Card>
  <Card title="🔑 Simple Auth" icon="key">
    API key authentication via the `X-API-Key` header; SDKs use `VR_API_KEY` (see [SDKs and API](/sdks/overview)). Self-hosted: override the API base URL with `VR_API_URL` or constructor options.
  </Card>
</CardGroup>

## How It Works

VoidRun architecture is built on MicroVMs for maximum isolation and security:

<Steps>
  <Step title="Create a Sandbox">
    Provision an isolated MicroVM with your preferred configuration and base image.
  </Step>
  <Step title="Execute Code">
    Run commands, upload files, and execute code in multiple languages securely.
  </Step>
  <Step title="Manage Lifecycle">
    Sandboxes run infinitely with auto-sleep. Resume instantly when needed.
  </Step>
  <Step title="Clean Up">
    Delete sandboxes when finished, or let them hibernate for later use.
  </Step>
</Steps>

## Use Cases

### AI Code Execution

Build AI agents that can safely execute generated code in isolated environments:

<CodeGroup>
```typescript TypeScript
import { VoidRun } from '@voidrun/sdk';

const vr = new VoidRun({ apiKey: process.env.VR_API_KEY });
const sandbox = await vr.createSandbox({ cpu: 2, mem: 2048 });

// Execute AI-generated code safely
const result = await sandbox.runCode(`
def calculate_fibonacci(n):
    if n <= 1:
        return n
    return calculate_fibonacci(n-1) + calculate_fibonacci(n-2)
    
print(calculate_fibonacci(10))
`, { language: 'python' });

console.log(result.stdout);
await sandbox.remove();
```

```python Python
from voidrun import VoidRun
import os

vr = VoidRun(api_key=os.environ.get("VR_API_KEY"))
sandbox = vr.create_sandbox(cpu=2, mem=2048)

# Execute AI-generated code safely
result = sandbox.run_code("""
def calculate_fibonacci(n):
    if n <= 1:
        return n
    return calculate_fibonacci(n-1) + calculate_fibonacci(n-2)
    
print(calculate_fibonacci(10))
""", language="python")

print(result.stdout)
sandbox.remove()
```

```bash CLI
# Create sandbox
voidrun create --cpu 2 --mem 2048

# Execute code (after sandbox is created)
voidrun exec <sandbox-id> --code "print('Hello from VoidRun')" --language python

# Clean up
voidrun delete <sandbox-id>
```

```bash cURL
# Create sandbox
curl -X POST https://platform.void-run.com/api/sandboxes \
  -H "X-API-Key: $VR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"cpu": 2, "mem": 2048}'

# Execute code (replace <sandbox-id> with actual ID)
curl -X POST https://platform.void-run.com/api/sandboxes/<sandbox-id>/exec \
  -H "X-API-Key: $VR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"command": "python3 -c \"print(\\\"Hello from VoidRun\\\")\""}'
```
</CodeGroup>

### Development Environments

Spin up ephemeral development environments with custom tooling:

<CodeGroup>
```typescript TypeScript
import { VoidRun } from '@voidrun/sdk';

const vr = new VoidRun({ apiKey: process.env.VR_API_KEY });
const sandbox = await vr.createSandbox({ 
  image: 'code',
  cpu: 4, 
  mem: 4096 
});

// Install dependencies and run tests
await sandbox.exec({ command: 'npm install && npm test' });
await sandbox.remove();
```

```python Python
from voidrun import VoidRun
import os

vr = VoidRun(api_key=os.environ.get("VR_API_KEY"))
sandbox = vr.create_sandbox(
    image="code",
    cpu=4,
    mem=4096
)
# Install dependencies and run tests
sandbox.exec("npm install && npm test")
sandbox.remove()
```

```bash CLI
voidrun create --template code --cpu 4 --mem 4096
voidrun exec <sandbox-id> --command "npm install && npm test"
voidrun delete <sandbox-id>
```

```bash cURL
curl -X POST https://platform.void-run.com/api/sandboxes \
  -H "X-API-Key: $VR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"image": "code", "cpu": 4, "mem": 4096}'
```
</CodeGroup>
