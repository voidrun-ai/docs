Voidrun exposes a REST API and also provides an official TypeScript SDK.

## Available SDKs

- TypeScript: @voidrun/sdk (official, maintained in this repo)
- Python: no official SDK in this repo yet

If you are not using the TypeScript SDK, you can call the REST API directly. See the API Guide for endpoint details.

## When to use which

- TypeScript SDK: fastest path for Node.js services and agent tooling
- REST API: any language or custom client
