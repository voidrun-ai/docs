---
title: Images
description: Manage base images used for sandboxes.
---

All endpoints below require X-API-Key and are prefixed with /api.

## List images

`GET /api/images`

## Create image

`POST /api/images`

```json
{
  "name": "ubuntu-22.04",
  "tag": "latest"
}
```

## Get image by ID

`GET /api/images/{id}`

## Get image by name

`GET /api/images/name/{name}`

## Delete image

`DELETE /api/images/{id}`
