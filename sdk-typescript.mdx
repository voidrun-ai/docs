## Install

```bash
npm install @voidrun/sdk
```

## Configure

The SDK reads configuration from environment variables by default.

```bash
export API_KEY="your-api-key-here"
export VOIDRUN_BASE_URL="http://localhost:33944/api"
```

## Quick start

```ts
import { VoidRun } from "@voidrun/sdk";

const vr = new VoidRun({});

const sandbox = await vr.createSandbox({ mem: 1024, cpu: 1 });
const result = await sandbox.exec({ command: 'echo "Hello from Voidrun"' });

console.log(result.data?.stdout);
await sandbox.remove();
```

## Common tasks

### Exec

```ts
await sandbox.exec({ command: "ls -la /root" });
```

### Files

```ts
await sandbox.fs.createFile("/tmp/hello.txt");
await sandbox.fs.uploadFile("/tmp/hello.txt", "Hello");
const buffer = await sandbox.fs.downloadFile("/tmp/hello.txt");
```

### PTY

```ts
const pty = await sandbox.pty.connect({
  onData: (data) => process.stdout.write(data),
});

pty.write("echo $SHELL\n");
pty.close();
```

### File watch

```ts
const watcher = await sandbox.fs.watch("/app", {
  recursive: true,
  onEvent: (event) => console.log(event.path, event.type),
  onError: (err) => console.error(err),
});

watcher.close();
```

## Code interpreter

The SDK includes a code interpreter for multi-language execution. See the SDK repository for supported languages and the interpreter API.
