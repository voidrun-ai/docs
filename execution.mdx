---
title: Execution
description: Run commands synchronously or stream output from sandboxes.
---

All endpoints below require X-API-Key and are prefixed with /api.

## Exec (synchronous)

`POST /api/sandboxes/{id}/exec`

```json
{
  "command": "ls -la /root",
  "timeout": 30,
  "env": {"PATH": "/usr/bin:/bin"},
  "cwd": "/root"
}
```

Notes:

- Empty or oversized commands are rejected
- timeout defaults to 30 seconds and is capped at 300 seconds
- If background is true, the server starts a background process

## Exec stream (SSE)

`POST /api/sandboxes/{id}/exec-stream`

Streams stdout, stderr, and exit events as Server-Sent Events.

## Session exec (agent session bridge)

`POST /api/sandboxes/{id}/session-exec`

```json
{
  "action": "create",
  "sessionId": "sess-...",
  "cols": 80,
  "rows": 24
}
```

Supported action values: create, exec, input, resize, close.

## Session exec stream (NDJSON)

`POST /api/sandboxes/{id}/session-exec-stream`

```json
{
  "sessionId": "sess-...",
  "command": "top -b -n 1"
}
```

Streams NDJSON chunks with Content-Type: application/x-ndjson.
