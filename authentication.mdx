## Register

`POST /api/register`

```json
{
  "name": "Admin",
  "email": "admin@example.com"
}
```

The response returns a user, default organization, and API key.

## Use the API key

All authenticated endpoints require the X-API-Key header.

```bash
curl http://localhost:33944/api/sandboxes \
  -H 'X-API-Key: hf_...'
```

WebSocket endpoints also accept apiKey as a query parameter for clients that cannot set headers.

## Key rotation

Use the org API key endpoints to create or revoke keys, and to toggle key activation.
